Projeto esta sendo desenvolvido na linguagem python utilizando o framework web django, e o banco de dados sendo trabalhado no PostgreSQL

para baixar os drivers necessarios é preciso utilizar os comandos:
Django: pip install django
PostgreSQL: python -m pip install psycopg2-binary --trusted-host pypi.org --trusted-host pypi.python.org --trusted-host files pythonhosted.org

e os para conectar com o banco de dados é necessaario ir para o arquivo C:\Metaphyta\Metaphyta\settings.py. Na linha 78 trocar as informações do banco de dados.
